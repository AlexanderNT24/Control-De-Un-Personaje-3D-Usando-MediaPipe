# Control De Un Personaje 3D Usando Mediapipe
El proyecto se basa en la creación de un personaje 3D que es controlado con inteligencia artificial, esto con la ayuda de la librería mediapipe, la cual reconoce y brinda puntos especiales en un rostro.
## Sobre El Proyecto
Para el desarrollo de nuestro proyecto usamos como IDE de desarrollo CodeBlocks para el diseño del personaje y Visual Studio Code para la programación del contról del personaje.
El proyecto hace uso de las siguentes tecnologías.
#### Para la creación del personaje 3D:
```
C++
OpenGL
Glut
```
#### Para el control del personaje:
```
Python 
Mediapipe
OpenCV
```
